# Конфиги и пути

- `/addons/sourcemod/plugins/` — `.smx` плагинов  
- `/addons/sourcemod/data/botmix/core.ini` — токены/настройки ядра  
- (опц.) `/addons/sourcemod/data/botmix/report.cfg` — лимиты, шаблоны причин  
- Логи: `addons/sourcemod/logs/botmix_*.log`
