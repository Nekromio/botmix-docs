# Система репортов (botmix_report)

**Назначение:** быстрые жалобы/обращения из игры в чат(ы) модерации с контекстом события.

---

**Файлы:**  
- `/addons/sourcemod/plugins/botmix/botmix_report.smx`  
- *(опц.)* интеграция с MaterialAdmin — подтягивание статусов, мутов и другой информации об игроке

---

## Потоки работы

**1. Игра → Чат**  
1. Игрок вводит `!report` или `sm_report`  
2. Открывается меню со списком игроков для выбора нарушителя  
3. Появляется меню выбора причины  
4. Жалоба отправляется во все привязанные чаты  

**2. Чат → Действие**  
1. В чате появляется интерактивное меню с информацией о репорте 
![Принять решение](/img/report.png)  
2. Модератор может выбрать действие (бан, мут, отложить решение)  

---

**Команды (сервер):**  
- `!report` — запустить меню репорта  
- `sm_report` — то же самое через консоль сервера  

---

**Требования:**  
- `botmix_core`  
- Активная платформа (VK и/или TG)
