# Платформа VK (botmix_vk)

**Назначение:** транспорт сообщений между сервером и беседой VK (через ядро).

**Ключевое:** ID беседы, права бота (админ), форматирование имён, кнопки/списки.

*Так же отвечает за вывод сообщений из VK чата

**Файл:** `/addons/sourcemod/plugins/botmix/botmix_vk.smx`  
**Зависит от:** `botmix_core`.
