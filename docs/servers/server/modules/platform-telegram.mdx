# Платформа Telegram (botmix_tg)

**Назначение:** связь сервера с группой/супергруппой TG (через ядро).

**Ключевое:** админ-права бота в TG, особенности форматирования, выбор чата через `!tg`.

*Так же отвечает за вывод сообщений из TG чата

**Файл:** `/addons/sourcemod/plugins/botmix/botmix_tg.smx`  
**Зависит от:** `botmix_core`.
