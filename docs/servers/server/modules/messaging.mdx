# Сообщения чат ↔ игра

Через ядро работают платформенные модули (VK/TG).

---

## Из чата в игру
- `!<id> Привет` — сообщение игрокам сервера `<id>`  
- `!<id>` — показать онлайн  

> Информация о факте отправки сообщения на игровой сервер в чате не отображается (чтобы избежать спама).

---

## Из игры в чат

**Игровой чат:**  
- `!vk <текст>` — выбрать беседу VK и отправить  
- `!tg <текст>` — выбрать чат TG и отправить  

**Консоль сервера:**  
- `vk <текст>` / `sm_vk <текст>` — выбрать беседу VK и отправить  
- `tg <текст>` / `sm_tg <текст>` — выбрать чат TG и отправить  

После ввода команды игроку выводится меню выбора чата для отправки.

---

**Замечания:**  
- Сообщения от замьюченных игроков не отправляются (при наличии MaterialAdmin).
